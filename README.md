# SQL Practice: 50 Query Challenge
---
This repository contains my solutions to a comprehensive 50-question SQL practice set. I used DBeaver to write, test, and optimize these queries against a relational database schema.
---
## 🚀 Key Skills Demonstrated
---
### 1. Advanced Joins
- Connecting Products, Sales, Customers, and Inventory tables using INNER, LEFT, and NATURAL joins.

### 2.Data Aggregation
- Summarizing business data using SUM(), COUNT(), and AVG().

### 3. Filtering Logic
- Implementing multi-level filters with WHERE (raw data) and HAVING (calculated data).

### 4. Time-Series Analysis 
- Extracting insights based on specific years and rolling time windows (e.g., "last 6 months") using INTERVAL and EXTRACT.

### 5. Subqueries & Set Theory
- Using NOT IN and NOT EXISTS to find "negative" data (like products that haven't sold).
